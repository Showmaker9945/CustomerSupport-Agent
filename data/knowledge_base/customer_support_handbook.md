# Customer Support Help Center

本文档用于演示智能客服项目中的帮助中心知识库。内容采用企业 SaaS 官方帮助文档写法，覆盖账户、账单、团队协作、安全、开发者能力、人工工单和服务时效等核心主题，适合用于文档摄入、结构化切分、混合检索和 HITL 流程验证。

## 文档使用说明

本帮助中心面向管理员、财务联系人、团队负责人和普通成员。若用户的问题涉及账户归属、权限变更、账单争议、退款审批或安全敏感操作，系统应优先给出标准政策，再根据风险等级触发人工审核或人工接管流程。

## 账户与登录

### 重置密码

如果用户忘记密码、登录密码失效，或需要重置登录凭证，可进入“设置 > 安全中心 > 密码”，点击“重置密码”。系统会向注册邮箱发送一封密码重置邮件，链接 24 小时内有效。若用户已无法访问注册邮箱，应先提交身份核验工单，经人工审核通过后再协助完成密码重置。

### 双重身份验证（2FA）

管理员和普通成员都可以在“设置 > 安全中心 > 双重身份验证”中开启 2FA。当前支持验证码 App 和短信两种方式。启用后，用户在输入密码后仍需完成第二步验证，适用于提升高权限账号和财务账号的安全性。

### 账户所有权转移

Enterprise 套餐支持账户所有权转移。发起转移后，当前负责人和目标负责人都需要完成确认。若转移涉及企业账单联系人、结算主体或高权限工作区，系统应要求人工审核后继续执行，以避免误操作或越权变更。

## 订阅与账单

### 支付方式

平台当前支持主流信用卡、PayPal，以及企业客户的对公转账。对于年付或月结客户，财务团队可根据合同配置发票结算或账期支付。若用户需要更改支付方式、付款方式或结算方式，建议先确认当前套餐、续费日期和未结清账单状态。

### 免费试用

新注册用户可申请 14 天 Pro 套餐免费试用，默认无需绑定信用卡。试用期结束后，用户可以选择升级到正式套餐，或继续使用免费版的受限功能。若企业客户需要更长试用期，通常需要联系销售团队单独审批。

### 取消订阅与退款规则

用户可在“设置 > 账单 > 订阅管理”中取消订阅、取消套餐或关闭当前订阅。取消后，服务会持续到当前计费周期结束。除法律法规另有要求或合同另有约定外，已使用周期通常不支持按天退款。若用户同时询问“取消后何时生效”与“是否会退款”，客服应同时解释计费周期结束时间和退款政策。

### 套餐升级、降级与席位变更

升级通常会立即生效，并按剩余周期按比例补差价；降级通常在下一个计费周期开始时生效。若用户调整席位数量，系统会根据当前套餐的席位上限和结算方式更新费用。客服在回答此类问题时，应优先确认当前套餐、席位使用数和下次续费时间。

### 账单异常与扣费说明

当用户反馈“扣费不对”“金额异常”“重复扣款”时，系统应先查询最近账单，并说明金额构成、账单周期、支付方式和费用明细。若问题涉及退款申请、重复扣费、对公结算、历史账单更正或人工核销，应创建“账单异常核查”工单，并进入人工审核或财务复核流程。

### 教育与公益优惠

经认证的教育机构和公益组织可申请专属折扣。申请时通常需要提交组织资质、域名邮箱或证明文件。优惠资格审核完成后，销售或客服团队会为符合条件的账户配置折扣方案。

### 存储配额与超限策略

当工作区存储使用达到 80% 和 95% 时，系统会发送提醒。超过上限后，新文件上传和部分同步写入会被暂停，但历史数据仍可读取。用户可以通过升级套餐、增加席位包或购买额外存储包恢复写入能力。

## 团队与权限

### 邀请团队成员

管理员可进入“设置 > 团队 > 成员”，点击“邀请成员”，填写邮箱并选择角色。系统会自动发送邀请邮件。若邀请对象长时间未收到邮件，应建议对方检查垃圾邮件、企业邮箱拦截规则或让管理员重新发送邀请。

### 角色与权限说明

平台默认提供管理员、编辑者和访客三类角色。访客只能查看授权内容；编辑者可以创建和修改内容，但不能调整系统级设置；管理员拥有成员管理、账单管理、安全配置和集成管理权限。涉及角色提升、移交管理员权限或批量权限变更时，建议记录审计信息并视风险触发人工审核。

## 安全与合规

### 数据安全

平台对静态数据采用 AES-256 加密，对传输链路采用 TLS 1.3。生产环境启用了访问控制、审计日志和最小权限原则。对于 Enterprise 客户，可按需提供更详细的安全说明、合规材料或架构说明文档。

### 数据驻留与部署区域

平台在美国、欧洲和亚洲提供数据中心或部署区域选择。用户可在注册或企业开通阶段选择首选区域；如需后续迁移，通常需要提交申请并评估数据迁移窗口、合规要求和服务影响范围。

## 开发者与集成

### API 访问

平台提供 REST API。管理员可在“设置 > 开发者”中创建 API Key，并按最小权限原则妥善保管。Pro 与 Enterprise 套餐默认支持 API 能力；如需更高频率配额、专用环境或专属限流策略，应联系销售或技术支持团队。

### 第三方集成

平台支持 Slack、Zapier、Google Workspace、Microsoft 365 等常见集成。部分 Enterprise 客户还可申请定制集成或专属回调策略。若集成失败，客服应先确认授权状态、回调地址、API Key 和第三方平台限制。

### 数据导出

用户可在“设置 > 数据 > 导出”中导出 CSV、JSON 或 PDF。对于大批量导出任务，系统会在后台生成导出文件，并通过邮件通知用户下载。若用户导出失败，建议先确认权限、导出范围和当前工作区数据量。

### 支持的设备与平台

Web 端支持主流现代浏览器；移动端支持 iOS 14+ 与 Android 10+；桌面端支持 macOS 11+ 和 Windows 10+。若用户反馈显示异常或功能不可用，应先确认浏览器版本、系统版本以及是否启用了企业网络代理或浏览器插件拦截。

## 支持与服务

### 联系客服

用户可以通过在线客服、帮助中心工单或支持邮箱联系支持团队。Pro 和 Enterprise 用户通常享有更高优先级支持。对于涉及结算、账户归属、安全事件或合同条款的问题，建议在首次响应时就明确说明是否需要人工介入。

### 工单响应时效（SLA）

免费版通常在 48 小时内响应，Pro 通常在 24 小时内响应，Enterprise 通常在 4 小时内响应。以上时效适用于工作日常规支持场景。若问题涉及账单异常、权限变更、账户转移、安全事件或高优先级故障，平台可能启用更快的内部升级流程。

### 人工审核与恢复流程

当用户明确要求创建账单异常工单、执行高风险写操作或请求人工升级时，系统应先进入人工审核。审核通过后，流程会在同一线程中恢复执行并继续创建工单或升级人工；审核拒绝时，系统应明确告知用户本轮未执行写操作，并给出下一步建议。该流程适合用于验证 LangGraph 的 interrupt/resume 能力以及 LangSmith 中的链路追踪效果。

## 产品更新与发布节奏

### 更新节奏

平台通常每周发布小版本优化，每季度发布一次较大的功能更新。绝大多数更新会以灰度方式上线，不影响用户正常使用。若用户询问“产品多久更新一次”或“最近有哪些变化”，客服应优先回答发布节奏，再视情况引导用户查看产品公告或版本说明。
